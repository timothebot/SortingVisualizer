More about algorithms

Quick Sort is a sorting algorithm. It is a divide and conquer algorithm. It picks an element as pivot and partitions the given array around the picked pivot. There are many different versions of quick sort. Click here to see one.

Bubble Sort is a sorting algorithm. It is a comparison sort. It compares the adjacent elements and swaps them if they are in wrong order. Click here to see one.

Selection Sort is a sorting algorithm. It is a comparison sort. It divides the input array into two parts, the sorted part and the unsorted part. It iterates over the unsorted array and finds the smallest element. It then swaps it with the element in the sorted array. Click here to see one.
